## Steps to get your project up and running

1. **Create an `env` file** using the example provided.
2. **Start the Docker environment**:

```bash
docker-compose up --build
Access the API in your browser or via tools like Postman:
Swagger documentation: http://localhost:21080/swagger/doc
Users endpoint: http://localhost:21080/users
User by ID: http://localhost:21080/users/{id}
Albums endpoint: http://localhost:21080/albums
Album by ID: http://localhost:21080/albums/{id}


Run tests (reports will be saved to backend/tests/_output/report.html):
